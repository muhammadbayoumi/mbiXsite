STEP 02

Unblock the File

Windows blocks files downloaded from the internet by default. You must unblock the .zip before extraction.

  1. Right-click mbiXaddin.zip
  2. Select Properties (or press Alt+Enter)
  3. In the General tab, check Unblock
  4. Click Apply, then OK
STEP 03

Extract & Run Installer

Extract the zip to any folder, then run the .vsto file.

  1. Extract mbiXaddin.zip
  2. Open the extracted folder
  3. Double-click mbiXaddin.vsto
  4. Click Install in the dialog
STEP 04

Open Excel — You're In

Open Microsoft Excel. A new mbiXaddin tab will appear in the Ribbon.
