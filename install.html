Install mbiX in Minutes

Follow the steps below to install mbiX on your Windows machine — no admin rights required.

Step 01

Requirements

Before installing mbiX, make sure your system meets these requirements.

📊
Microsoft Excel
2016 / 2019 / 2021 / 365
⚙️
.NET Framework
v4.7.2 or higher
🪟
Windows
10 / 11 (64-bit)
🌐
Internet
For first-time sync only
Step 02

Download Setup

Download the latest installer from the official Releases page on GitHub. Always grab the .msi file — it handles registration automatically.

⬇ mbiXaddin-v2.0.msi
Downloads
# Latest release
file: mbiXaddin-v2.0.msi
size: ~12 MB
checksum: SHA256 verified
# What gets installed
→ %APPDATA%\mbiX\
→ mbiXdata.sqlite3
→ Excel Add-in registration
Step 03

Run Installer

1

Close Excel

Make sure all Excel windows are closed before running the installer.

2

Run the .msi

Double-click the installer file. Follow the wizard — defaults are fine.

3

Open Excel

A new "mbiX" tab appears in the Ribbon. You're ready to sync.

Step 04

Configure Google Sheet

mbiX needs 5 system tables in your Google Sheet to know what to sync. Create them once — then add your data as needed.

View Schema Docs →
SYS_DEFINITIONS SYS_SCHEMA_RULES SYS_DATA_SOURCES SYS_DATA_MAP SYS_CONFIG_PRESETS
