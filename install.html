STEP 02

Unblock the File

Windows blocks files downloaded from the internet by default. You must unblock the .zip before extraction — otherwise the .vsto installer will fail silently.

  1. Right-click mbiXaddin.zip
  2. Select Properties (or press Alt+Enter)
  3. In the General tab, check Unblock
  4. Click Apply, then OK

STEP 03

Extract & Run Installer

Extract the zip to any folder, then run the .vsto file. Office will show a publisher confirmation — click Install.

  1. Extract mbiXaddin.zip
  2. Open the extracted folder
  3. Double-click mbiXaddin.vsto
  4. Click Install in the dialog

Help

Troubleshooting

🚫
Tab not appearing in Excel
Did you Unblock the zip BEFORE extracting? If not — delete the extracted folder, re-download, unblock, and try again.
🔒
"Cannot install" error
Make sure all Excel windows are closed before running .vsto. .NET 4.7.2+ must be installed.
🛡
Antivirus blocks installer
Add the extracted folder to your antivirus exclusion list, or temporarily disable real-time protection during install.
