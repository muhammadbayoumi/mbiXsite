Troubleshooting

Tab not appearing in Excel

Did you restart Excel after installation? Close all Excel windows completely, then reopen Excel. The mbiX tab should appear in the Ribbon.

"Cannot install" error

Did you unblock the zip file BEFORE extracting? Right-click the zip → Properties → check "Unblock" → Apply → then extract and run the installer again.

Antivirus blocks installer

Add the extracted folder to your antivirus exclusion list.
